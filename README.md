A Node.js project that uses Google's Gemini AI API. 

1. Install the dependencies.
2. Get a Gemini API key from Google.
3. Enter this key (as a string) as the value of Gemini_API_Key in the .env file:
Gemini_API_Key='yourKeyHere'
4. Enter a dream that you would like to have interpreted as the prompt string.
5. Run node.
6. Reflect.
